# Roadmap

## Overview

TinyTalk is intentionally built in small, manageable phases.

Each phase should produce a working result before moving to the next.

The goal is to build a complete MVP as quickly as possible while keeping the project simple.

# Future Ideas

These features are intentionally excluded from the MVP.

Possible future improvements include:

- Favorite topics
- Daily recommended lesson
- Pronunciation scoring
- Learning statistics
- AI-generated conversations
- Additional content packs

---

# MVP Goal

The MVP is complete when the parent can:

- Open the website.
- Search for a topic.
- Complete a voice conversation.
- Continue learning on another device.
- Enjoy a simple and reliable daily speaking experience.

---

# Development Philosophy

Always finish the current phase before starting the next one.

Avoid adding new features until the MVP is complete.

A small, finished product is better than a large, unfinished one.
